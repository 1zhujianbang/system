# MarketLens page branch update

`page`分支的当前版本主要更新了以下内容：
v1.1
- 图谱悬停卡片增强：支持驻留、摘要/事件类型、涉及实体完整展示并可点击聚焦
- GE 图新增事件类型聚焦筛选
- EE 图新增共现边悬停卡片（按时间排序展示关系），并支持与实体关系边并行展示
- GET 时序图节点间距调整（提升可读性）
- 修复悬停卡片渲染为小框、悬停事件不触发、悬停高亮但卡片不显示等问题
- 实体节点“出现频次”字段修复（从存储导出补齐 count）
- EE 物理引擎参数调整：增加斥力避免吸附，并通过稳定化参数让布局更快收敛停止

这些更新都是在 `page` 分支上完成的，等待测试完全通过后将合并到主分支。

更多新增功能需要先向该分支合并，贡献者需要提交PR用于新增功能
