# MarketLens feature branch update

feature分支的当前版本主要更新了以下内容：

## 1. **测试基础设施完善**
- 为所有新增模块创建了完整的单元测试和集成测试
- 覆盖了 Business 层、Adapter 层和 Pipeline 层的所有新功能

## 2. **新增功能模块**
- **因果网络分析模块**：实现了基于 GDELT 数据的因果关系构建
- **社区检测模块**：使用 Louvain 算法进行社区发现和观点阵营识别
- **动态仿真引擎**：支持事件流驱动的动态仿真和影响传播
- **反事实推理模块**：集成 dowhy 库进行因果干预分析

## 3. **技术架构升级**
- **存储迁移**：从 SQLite 迁移到 Neo4j 图数据库
- **LLM 增强**：集成 GraphRAG 技术提升检索增强生成能力
- **分布式处理**：通过 Celery 实现 Pipeline 的分布式执行

## 4. **部署优化**
- 完善了 Docker 容器化部署配置
- 支持生产环境的多实例部署和负载均衡
- 集成了 Nginx 反向代理和 SSL 证书配置

## 5. **性能优化**
- 实现了 Redis 缓存机制
- 优化了图数据库查询性能
- 添加了慢查询分析工具

这些更新都是在 `feature` 分支上完成的，等待测试完全通过后将合并到主分支。

更多新增功能需要先向该分支合并，贡献者需要创建一个该分支的子分支用于新增功能
